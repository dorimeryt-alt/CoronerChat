1. Введение

CoronerChat — десктопный кабинет стримера: единая лента Twitch, VK Video Live, Kick, YouTube и Rutube, донаты DonationAlerts / MemeAlerts, модерация, OBS overlay и управление эфиром. Настройки шоу (чеклист, пресеты, алерты, favorites, resend, статистика, достижения) — вкладка «Эфир».

Рекомендуется пройти разделы 2–4 и «Сообщения», затем вкладку «Эфир». В приложении есть «Помощник настройки» — платформы и фичи по шагам.

2. Интерфейс при входе

  • Шапка — часы, масштаб шрифта, уведомления, ⚙. «Зрителей всего» и счётчики по площадкам. Блок OBS-телеметрии (если включён).
  • Чеклист «готов к эфиру» — чипы TW/VK/Kick/OBS и пресеты / Фокус (F10).
  • Центр событий — донаты, рейды, подписки, статусы. Только в приложении, не в OBS overlay.
  • Чат — общая лента. Клик по нику — модерация; по тексту — выделение; ПКМ — меню.
  • Панель ввода — поля площадок (включаются во вкладке «Вид»). Slash-команды Twitch — как на сайте.
  • Tools — опросы, ставки, клипы, фильтр, модерация (для владельца канала).

3. Быстрый старт

  1. ⚙ → Twitch → канал → «Подключить» → «Авторизоваться через Twitch».
  2. Мультистрим: вкладки VK, Kick, YouTube, Rutube по необходимости.
  3. Или Помощник настройки («О приложении» / первый запуск).
  4. Вид → скопировать Browser Source → вставить в OBS.
  5. Опционально: OBS WebSocket; Донаты; Эфир → алерты / goal / vote.

Подробнее по установке: Install.

4. Площадки чата

Twitch. IRC читается сразу; OAuth нужен для отправки, опросов, предсказаний, рейдов, клипов, title/category, Shield Mode и модерации API. 7TV подгружается автоматически. Channel Points / Hype Train / Watch Streak — во вкладке «Уведомления».

VK Video Live. URL live.vkvideo.ru/… → Подключить. Для отправки — вход через VK. Reply: REPLY / ПКМ / R.

Kick. Slug канала → Подключить. OAuth — для отправки и бейджей.

YouTube. «Войти через Google» или API key / browser chat. Учитывайте квоты Google Cloud. В РФ часто нужен VPN.

Rutube. URL эфира → чтение через poll API. Отправка — cookie через встроенное окно входа.

Карточки-сводка: Platforms.

5. DonationAlerts и MemeAlerts

DonationAlerts — OAuth во вкладке «Донаты»; события в чате и центре событий; при включённых алертах UX — очередь ?alerts=1 и прогресс ?goal=1.

MemeAlerts — целиком «Ссылка для OBS» с ?obsToken= (Socket.IO + JWT). Фильтр «Донаты» показывает оба источника. Мин. сумму и скрытие можно задать отдельно.

6. OBS WebSocket и overlays

WebSocket. Интеграции → URL обычно ws://127.0.0.1:4455 + пароль OBS → «Подключить OBS». Сцены, Start/Stop Stream/Record, телеметрия, компактный док.

Browser Sources:

  • ?overlay=1 — чат для зрителей (~400×900)
  • ?musicOverlay=1 — Now Playing (системный медиасеанс Windows или bridge JSON)
  • ?alerts=1 · ?goal=1 · ?vote=1 · ?label=1&key=…
  • ?recap=1 — итоги эфира для зрителей
  • ?modview=1 — LAN-режим для модеров

Кнопка «Копировать все OBS URL» в UX-пакете копирует chat + music + alerts + goal + vote одной пачкой.

7. Сообщения: выделение, меню, хоткеи

  • Клик по тексту — выделение (цель для deck/мода). Повторный клик снимает.
  • Клик по нику — PIN, DEL, INFO, таймауты, BAN.
  • ПКМ — копировать, цитата, избранное, скрыть автора, карточка, ответ VK, санкции, resend.
  • Hover — COPY · HIDE · INFO (+ REPLY для VK).

Хоткеи (фокус не в поле ввода): C копировать · Shift+C в поле · H скрыть · I/F карточка · Delete удалить · B бан · T таймаут 10м · R reply VK · Esc сброс · F10 фокус-режим.

8. Эфир / UX-пакет

Все настройки шоу: ⚙ → Эфир. Поиск — в шапке панели настроек.

  • Пресеты TW / Мульти / Полн. — показывают/скрывают composer’ы (не рвут подключения).
  • Фокус / F10 — лента + активное поле ввода.
  • Favorites, keywords, цензор, soft-confirm — лента и модерация.
  • Resend — ретрансляция входящих в другие чаты.
  • Song request — !sr / !song, очередь, Skip.
  • Safe mode / Always on top — сброс темы и окно поверх.

9. Модерация и карточка пользователя

Карточка (INFO / I): история, заметки, журнал мод-действий, timeout/ban/unban. Центр модерации — очередь AutoMod/suspicious и шаблоны санкций. Soft-confirm — в UX-пакете.

10. Фильтры, подсветка, авто-ответы

Фильтр чата (tools): скрыть платформу, поиск. UX-фильтры: избранные, сабы, боты, reply. Правила подсветки (Общие) — keyword /regex/. Авто-ответы — триггер + текст. Справочник @-алиасов и списки скрытых авторов.

11. Стили, темы и локализация

Вид — пресеты (Base, Relay, Ember, Eru, Retro, Control Room, Arcade, Brutalist, Broadcast, Электронная книга и др.), шрифт, акцент, плотность, daltonism. Overlay может следовать приложению или задаваться отдельно.

Язык UI: Общие → ru / en / ro / de / fi. Смена языка сохраняется и применяется без поломки контролов (2.7.5+).

12. Сеть, планшет, deck

Интеграции — LAN-адрес, планшетный URL, QR. Одна Wi‑Fi; при блокировке — правило firewall. WebSocket telemetry и /api/state — для внешних скриптов. Ulanzi / Stream Deck — через /api/deck/* (на не-localhost нужен CORONERCHAT_DECK_TOKEN). Zapret (Windows) — опциональный обход DPI для групп доменов.

13. Статистика и достижения

Вкладка Эфир / Статистика: история сессий, категории/игры, топы чата и донатов (сессия / месяц / год / всё время). Достижения — полка в приложении (не в OBS). Публичный recap: ?recap=1.

14. Обновления

Канал dorimeryt-alt/CoronerChat, проверка при запуске. «Скачать и установить», «Пропустить версию», «Напомнить позже». С 2.7.5 — только GitHub Latest (без ложного 2.35.x). Подробнее: Update.

15. Безопасность оператора

По умолчанию HTTP только на 127.0.0.1. Токены в CoronerChat-data — не шарьте экспорт настроек. Не включайте 0.0.0.0 без deck-токена и сегментации сети. Полный compliance-раздел: Security.

16. Частые вопросы

  • Чат не коннектится? Ник/URL, статус OAuth, чеклист эфира, «Статус платформ».
  • Rutube не шлёт? Сначала cookie-вход во встроенном окне.
  • Хоткеи молчат? Фокус не в поле ввода, сообщение выделено, настройки закрыты.
  • Алерты пустые в OBS? Отдельный Browser Source с нужным query; DA подключён; очередь не на Pause.
  • OBS WS? Server в OBS, пароль, 127.0.0.1:4455.
