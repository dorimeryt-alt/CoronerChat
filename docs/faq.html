FAQ

Коротко про скачивание, странные версии, данные и донат. Разверни карточку.

Где скачать последнюю версию?

github.com/dorimeryt-alt/CoronerChat/releases/latest

Почему предлагали 2.35.x вместо 2.7.x?

В Releases лежит архив со старой нумерацией (2.25–2.35). Они выглядят «больше» по semver, но это не актуальная линия.

С 2.7.5 приложение читает GitHub Latest и игнорирует legacy-теги. Актуальная сборка — 2.7.10. Если сидишь на 2.7.4 — один раз поставь Latest вручную.

Пропали настройки после обновления?

Проверь папку CoronerChat-data рядом с приложением. Её нельзя удалять при переустановке, если нужны токены и UI.

Имена файлов с точками вместо пробелов?

Так GitHub иногда нормализует assets. CoronerChat ищет .exe с Setup и CoronerChat в имени — оба варианта ок.

При смене языка «ломается» интерфейс?

Исправлено в 2.7.5: auto-i18n больше не затирает чекбоксы и вложенную разметку. Обновись до Latest.

Где исходный код?

В этом репозитории — только установщики и документация. См. описание модели распространения.

Языки интерфейса?

Настройки → Общие: русский, English, Română, Deutsch, Suomi.
