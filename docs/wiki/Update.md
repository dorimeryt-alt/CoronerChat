# Обновления

Источник канала: `dorimeryt-alt/CoronerChat`. Актуальная линия: **2.7.10**.

1. Настройки → Обновления (или баннер при запуске).
2. Автопроверка по желанию.
3. **Скачать и установить** / **Пропустить эту версию** / **Напомнить позже**.

Приложение читает релиз GitHub **Latest**, а не «максимальный» semver из архива (старые теги 2.25–2.35 не предлагаются поверх линии 2.7).

Сайт: https://dorimeryt-alt.github.io/CoronerChat/update.html
