# CoronerChat

**CoronerChat** — десктоп-приложение для стримера: чаты нескольких площадок в одной ленте, донаты, OBS overlay, алерты, модерация, статистика и достижения.

## Этот репозиторий

Публичный **канал обновлений** (GitHub Releases) + документация.

Приложение проверяет `dorimeryt-alt/CoronerChat` при запуске и предлагает Setup.exe.
