# CoronerChat

**CoronerChat** — десктоп-приложение для стримера: чаты нескольких площадок в одной ленте, донаты, OBS overlay, алерты, модерация, статистика эфиров и достижения.

## Этот репозиторий

Публичный **канал обновлений** (GitHub Releases) + краткая документация.

Приложение в настройках смотрит на `dorimeryt-alt/CoronerChat` и предлагает скачать `Setup.exe` новой версии.
