Площадки

Все источники сходятся в одну ленту. Донаты — отдельная вкладка, overlay — в OBS.

Чаты

ПлощадкаЧто даётКак подключить
TwitchЧат, отправка, мод, рейды, категория/тайтл, Shared ChatDevice Flow / OAuth во вкладке Twitch
VK Video LiveЧатOAuth VK во вкладке VK
KickЧатКанал + OAuth при необходимости
YouTube LiveЧатGoogle OAuth / Live URL (в РФ часто нужен VPN)
RutubeЧат / liveURL стрима во вкладке Rutube

Донаты

СервисТранспортНастройка
DonationAlertsOAuth + HTTP pollClient ID/Secret → авторизация в браузере
MemeAlertsSocket.IO + JWTСкопируй «Ссылку для OBS» целиком (?obsToken=)
Фильтр чата «Донаты» показывает события и DonationAlerts, и MemeAlerts. Мин. сумму и скрытие строк можно задать отдельно для каждого источника.

OBS

  • Browser Source → локальный URL приложения + ?overlay=1
  • Итоги эфира для зрителей: ?recap=1
  • Опционально: WebSocket telemetry OBS в настройках интеграций
