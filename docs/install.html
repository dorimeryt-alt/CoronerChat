Установка

Один Setup.exe — и кабинет оператора готов. Данные эфира живут отдельно от бинарника.

Быстрый старт

  1. Открой Releases → Latest.
  2. Скачай файл установщика: CoronerChat.Setup.x.y.z.exe (GitHub может заменить пробелы на точки).
  3. Запусти NSIS-мастер и следуй шагам.
  4. После установки открой CoronerChat и подключи площадки во вкладках настроек.
CoronerChat-data — папка рядом с приложением: app-state, токены, логи, кэш. При обновлении NSIS она сохраняется. Не удаляй её, если не хочешь сбросить настройки.

Системные требования

  • Windows 10 / 11, x64
  • Сеть до Twitch / VK / Kick / YouTube / Rutube / DA / MemeAlerts (где нужно — VPN)
  • OBS Studio — для Browser Source overlay

Первый запуск

  1. Настройки → язык интерфейса (ru / en / ro / de / fi).
  2. Подключи Twitch (и другие площадки по необходимости).
  3. Донаты: DonationAlerts (OAuth) и/или MemeAlerts (ссылка OBS с ?obsToken=).
  4. В OBS добавь Browser Source на локальный URL приложения с ?overlay=1.

Portable vs Setup

Ранние сборки (1.x) — один portable .exe. Актуальная линия — NSIS Setup. Оба варианта держат данные в CoronerChat-data.
