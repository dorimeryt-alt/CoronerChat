Установка

  1. Открой Releases → Latest.
  2. Скачай CoronerChat.Setup.x.y.z.exe.
  3. Запусти NSIS-установщик.

Данные хранятся в CoronerChat-data рядом с приложением и сохраняются при обновлении.

Windows 10/11 x64. Для чатов нужны аккаунты площадок; для overlay — OBS Browser Source.
