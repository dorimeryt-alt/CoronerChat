Stage 01 · Boot

Установка

Один Setup — и кабинет оператора готов. Данные эфира живут отдельно от бинарника.

01 · Download

Скачай Latest

Открой Releases → Latest и возьми CoronerChat.Setup.x.y.z.exe.

02 · Install

Запусти NSIS

Мастер установки Windows x64. После финиша ярлык CoronerChat появится в меню Пуск.

03 · Connect

Подключи площадки

⚙ → Twitch / VK / Kick / YouTube / Rutube. Донаты и OBS — следующим заходом.

04 · Overlay

Добавь в OBS

Вид → скопируй Browser Source URL с ?overlay=1 → вставь в OBS (~400×900).

Portable mode

Портативная сборка

Если в Latest опубликован файл без Setup в имени, это портативная сборка: помести EXE в отдельную папку с правом записи и запускай оттуда без установки.

Не запускай портативную сборку из архива, временной папки или Program Files. Храни рядом папку CoronerChat-data, переноси их вместе и перед заменой EXE делай резервную копию данных.

Save slot

CoronerChat-data

Папка рядом с приложением: app-state, токены, логи, кэш. При обновлении NSIS она сохраняется.

Не удаляй CoronerChat-data, если не хочешь сбросить настройки и авторизации. Перед переносом ПК — скопируй её целиком.
