Обновления

Канал зашит в приложение: dorimeryt-alt/CoronerChat. Проверка при каждом запуске.

Из приложения (рекомендуется)

  1. Настройки → Обновления — или баннер на главном экране.
  2. «Проверить сейчас» / автопроверка при старте (галочка включена по умолчанию).
  3. Скачать и установить — качается Setup во временную папку и запускается NSIS.
  4. Либо Пропустить эту версию (больше не показывать её) / Напомнить позже (только до перезапуска).
С версии 2.7.5 приложение берёт релиз, помеченный на GitHub как Latest. Архивные теги вроде 2.35.1 больше не перебивают линию 2.7.x.

Вручную

Скачай Setup с Releases и установи поверх. Папка CoronerChat-data останется.

Каналы релизов

  • Только стабильные — GitHub Latest (по умолчанию).
  • Включая prerelease — если нужно ловить тестовые теги.

Публикация (для сборщиков)

npm run pack:win:installer
npm run release:github

Notes берутся из changelog в build/installer.nsh.
