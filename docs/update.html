Stage 02 · Patch

Обновления

Канал зашит: dorimeryt-alt/CoronerChat. Проверка при каждом запуске — без лишних настроек.

01 · Check

Баннер или вкладка

При старте — баннер на главном экране. Или ⚙ → Обновления → «Проверить сейчас».

02 · Install

Скачать и установить

Setup качается во временную папку и запускает NSIS. После мастера — перезапуск CoronerChat.

03 · Skip

Пропустить версию

«Пропустить эту версию» — больше не показывать её. «Напомнить позже» — только до перезапуска.

04 · Manual

Вручную с GitHub

Скачай Setup с Releases и поставь поверх. Data-папка останется.

Latest lock

Почему не 2.35.x

С 2.7.5 приложение читает GitHub Latest и отсекает legacy-архив (2.25–2.35), который ломал semver-сравнение.

Latest feed stable by default legacy tags ignored
Сидишь на 2.7.4 и видишь странный апдейт? Один раз поставь 2.7.5+ вручную с Releases — дальше канал будет чистым.
