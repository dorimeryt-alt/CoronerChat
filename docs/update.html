Обновления

Источник зашит: dorimeryt-alt/CoronerChat.

  1. Настройки → Обновления (или баннер при запуске).
  2. «Проверить» / автопроверка при старте.
  3. «Скачать и установить» или «Пропустить эту версию».

Приложение берёт релиз, помеченный на GitHub как Latest (не максимальный semver из архива — старые теги вроде 2.35.x игнорируются для линии 2.7).
