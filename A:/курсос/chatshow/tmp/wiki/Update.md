# Обновление

## Из приложения (рекомендуется)

1. **Настройки → Обновления**
2. Репозиторий по умолчанию: `dorimeryt-alt/CoronerChat` (можно не менять)
3. **Проверить** — сравнение версии с GitHub Releases
4. **Скачать и установить** — качается Setup во временную папку и запускается установщик

После установки перезапусти CoronerChat. `CoronerChat-data` не затирается.

## Вручную

Скачай Setup с [Releases](https://github.com/dorimeryt-alt/CoronerChat/releases) и установи поверх.

## Ветка feed

Кодовая ветка зеркала: `releases/update-feed`. Для проверки обновлений приложению достаточно **Releases**, не клона репо.
