# Установка

1. Открой [Releases](https://github.com/dorimeryt-alt/CoronerChat/releases/latest).
2. Скачай файл установщика: `CoronerChat.Setup.x.y.z.exe` (или `CoronerChat Setup x.y.z.exe`).
3. Запусти установщик (NSIS), следуй шагам.
4. Данные приложения хранятся в папке **`CoronerChat-data`** рядом с установкой / portable — при обновлении она **сохраняется**.

## Старые версии (1.x)

Ранние сборки выложены как **portable** `.exe` (один файл без Setup). Запуск — двойной клик; рядом появится `CoronerChat-data`.

## Системные требования

- Windows 10/11 x64
- Для чатов: аккаунты площадок и OAuth из настроек приложения
- Для overlay: OBS Browser Source на локальный URL приложения
